1. Instal VMWare.
	- Minimum:
		- 2GB Memory & 20GB Storage. (Not recommended to git clone large github repo)
	- Recommended:
		- 4GB Memory & 40-50GB Storage.

2. Tested on Linux Distro.
	- Ubuntu 18 - 22. (Compatible & Stable)

3. Complete the installation by setting the user credentials.
	- Username: #Insert your username
	- Password: #Insert your password

4. Update & install packages needed.
	- Prerequisite:
		- sudo apt-get update
		- sudo apt-get upgrade -y
		- sudo apt-get install gcc-arm-none-eabi

	- Optional:
		- sudo apt-get install git vim

	- Ai-Thiker:
		- sudo apt-get install python3 python-pip

5. Clone the git repo to the system.
	- git clone https://github.com/ambiot/ambd_sdk.git

6. Change permission to allow the user to r/w/x every files.
	- chmod -R 777 ./

7. Go into project hp and project lp directory, then run make command.
	- cd ..path/project_lp
	  make all
	- cd ..path/project_hp
	  make all
